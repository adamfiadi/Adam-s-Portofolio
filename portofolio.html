Here are some projects that I've done for the past couple of months

Bajoe.id

Bajoe.id is my first web-based project that me and a few colleagues had done in our 3rd Semester. In description, it is a fictional ecommerce clothing company we built in reference to famous online shops like shopee, tokopedia, and bukalapak.

My team consists of 5 members each having their own tasks in partaking this project. I was assigned to handle the backend part of the website, finishing the final assembly, and even helped a few errors in the frontend part. The backend tools I used in this project are nodejs (integration), express (server), and mysql (databse) in order to store the user datas.

Github : https://github.com/adamfiadi/Web-Prog-Final-Project

Enchanting

Enchanting is my team's final project website based on an event called Joints. Joints is a technology event held by computer science majors from UGM. It has a few subevents called Grand Launching, Jointscamp, and Techtalk. Wanting to know more about the IT World, I signed up to be a participant of the Frontend Division in Jointscamp.

The goal for this final project is to build a website using only html, css, and javascript (frontend frameworks allowed). Every participant were required to finsih the final project in teams of 3 (chosen randomly by the commitee). Therefore, my team decided to make an online food catering website that implemented all the materials we learned at the camp. In this opportunity, I was assigned to design the UI/UX and make the frontend as well. The tools and framworks I used in this project are bootstrap, jquery, and figma (for designing the UI/UX).
